Jeeliz Camera Exposure Adjuster

You have an important video conference? Come here and your webcam will be automatically adjusted! It is particularly efficient when:

  • There is a strong backlight so your face is hidden in the shadow,
  • The ambient luminosity of the room is too low.


When the face is not detected, the exposure changes following a sinuso√Ød. As soon as your face is detected, the exposure is tuned so that the average luminosity of your face is 50%. After ajusting the luminosity, you can exit this page.

This demo requires Image Capture API in order to adjust the camera exposure. It is currently implemented in Google Chrome web browser. You can check the compatibility table here: developer.mozilla.org/en-US/docs/Web/API/ImageCapture. It is also necessary that the camera drivers allow to change the exposure (this is not possible on some cheap webcams). If it does not work on your computer, you can still check the Youtube screenshot video.

The exposure is changed using Jeeliz Exposure Controller.
